# nhentai-favorites

### how to use?
`pip install -r ".\requirements.txt"`


Rename_me_set.yaml (rename)-> set.yaml  
enter your cookie  
open nfavorites.py

### how to get my cookie?

open https://nhentai.net/favorites/  
press F12  
switch to network menu  
find favorites/  
copy cookie to set.yaml
